## ğŸ”§ Installation

1. **Clone the repository**

```bash

```

2. **Navigate to the project directory**

```bash
cd Realtime-Weather-and-Prayer-Times-APP
```

3. **Get API Keys**
   - OpenWeather API: Register at [OpenWeatherMap](https://openweathermap.org/api)
   - Prayer Times API: Free access at [Aladhan API](https://aladhan.com/prayer-times-api)

4. **Configure API Keys**
   - Update `js/main.js` with your API keys
   - Ensure secure storage of sensitive data

5. **Launch the application**
   - Open `index.html` in a modern browser
   - Or use a local server (e.g., VS Code Live Server)
