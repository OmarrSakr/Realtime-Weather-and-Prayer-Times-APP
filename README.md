The app features an intuitive interface with **automatic data updates**, **location-based services**, **comprehensive weather forecasting**, and **complete Arabic language support**.

## 🚀 Features

- 🌦️ **Real-Time Weather**: Live weather data with current temperature, conditions, and forecasts.
- 🕌 **Daily Prayer Times**: Accurate prayer timings (Fajr, Dhuhr, Asr, Maghrib, Isha) based on location.
- 📍 **Location Services**: Auto-detect user location or search by city name.
- 📅 **5-Day Forecast**: Extended weather predictions with daily summaries.
- ⏰ **Hourly Forecast**: Detailed hourly weather breakdown for better planning.
- 🌅 **Sunrise & Sunset**: Precise timing for sunrise and sunset with visual indicators.
- 💨 **Air Quality Index**: Real-time AQI data with pollutant levels (PM2.5, PM10, CO).
- 📱 **Responsive Design**: Optimized interface for desktop, tablet, and mobile devices.
- 🎨 **Modern UI**: Clean, intuitive design with Font Awesome icons and Google Fonts.
- 🔄 **Auto-Updates**: Automatic data refresh for current information.
- 🔔 **Prayer Notifications**: Audio alerts 5 minutes before prayer time (NEW).
- 🌙 **Dark Mode**: Toggle between light and dark themes (NEW).
- 🌍 **Multi-Language Support**: Full Arabic support with plans for additional languages.
- 🧭 **Qibla Compass**: Shows Qibla direction with green arrow and Arabic description (NEW).

## 🛠️ Website Components

### **1. Header Section**
- 🏷️ **Site Title**: "Weather & Prayer Times" with relevant weather and prayer icons
- 🔍 **Search Interface**:
  - **City Input Field**: Enter city name for weather lookup
  - **Search Button**: Fetch weather data for specified location
  - **Current Location Button**: GPS-based weather and prayer times retrieval

### **2. Current Weather Section**
- 🌡️ **Live Temperature**: Real-time temperature display in Celsius
- 🎨 **Weather Icon**: Visual weather representation (sunny, rainy, cloudy, etc.)
- 📅 **Additional Info**:
  - Current date and time
  - City name and country
  - Weather description

### **3. 5-Day Forecast Section**
- 📊 **Forecast Cards**: Each card displays:
  - Weather icon for the day
  - Expected high/low temperatures
  - Day name and date
  - Weather conditions summary

### **4. Today's Highlights**
- 🌬️ **Air Quality Index (AQI)**: Comprehensive air quality readings
- **Pollutant Data**: PM2.5, PM10, CO, NO2, SO2, O3 levels
- 🌅 **Sunrise & Sunset**: Accurate timing with custom icons
- 📊 **Weather Metrics**:
  - Humidity percentage
  - Wind speed and direction
  - Atmospheric pressure
  - Visibility distance
  - "Feels like" temperature

### **5. Hourly Forecast Section**
- ⏰ **Time-Based Forecast**: Weather by hour (6 AM, 9 AM, 12 PM, etc.)
- 🌡️ **Hourly Details**:
  - Weather icons
  - Temperature predictions
  - Precipitation chances

### **6. Prayer Times Section**
- 📅 **Islamic Calendar**: Current Hijri date and day
- 🕌 **Prayer Cards**: Individual cards for each prayer:
  - **Fajr** (Dawn Prayer)
  - **Dhuhr** (Midday Prayer)
  - **Asr** (Afternoon Prayer)
  - **Maghrib** (Sunset Prayer)
  - **Isha** (Night Prayer)
- 🎯 **Accurate Timings**: Location-based precise prayer times
- 🧭 **Qibla Compass**: Shows Qibla direction with green arrow (Updated)

### **7. External Resources**
- 🎨 **Font Awesome Icons**: Enhanced visual elements
- 🔤 **Google Fonts**: Typography (Roboto, Montserrat)
- 🖼️ **Local Assets**: Weather images and prayer icons stored in `assets/Imgs`

## 🔑 API Integration

### **OpenWeather API**
- **Current Weather**: `https://api.openweathermap.org/data/2.5/weather`
- **5-Day Forecast**: `https://api.openweathermap.org/data/2.5/forecast`
- **Air Quality**: `https://api.openweathermap.org/data/2.5/air_pollution`
- **Features**: Temperature, humidity, wind, pressure, UV index

### **Aladhan Prayer Times API**
- **Endpoint**: `https://api.aladhan.com/v1/timings`
- **Method**: Islamic Society of North America (ISNA)
- **Features**: Location-based accurate prayer times, Hijri calendar
- **New Addition**: Qibla compass using `https://api.aladhan.com/v1/qibla`

## 🔧 Installation

1. **Clone the repository**

```bash
git clone https://github.com/OmarrSakr/Realtime-Weather-and-Prayer-Times-APP.git
```

2. **Navigate to the project directory**

```bash
cd Realtime-Weather-and-Prayer-Times-APP
```

3. **Get API Keys**
   - OpenWeather API: Register at [OpenWeatherMap](https://openweathermap.org/api)
   - Prayer Times API: Free access at [Aladhan API](https://aladhan.com/prayer-times-api)

4. **Configure API Keys**
   - Update `js/main.js` with your API keys
   - Ensure secure storage of sensitive data

5. **Launch the application**
   - Open `index.html` in a modern browser
   - Or use a local server (e.g., VS Code Live Server)
